# IPKN-ML Course

- Lecture slides
- Seminar notebooks

## Setup

Use [`uv`](https://docs.astral.sh/uv/getting-started/installation/) or `pip` for dependency management.

Before using notebooks syncronize project:

```sh
uv sync
```

or without `uv`:

```sh
pip install -r rquirements.txt
```

Open folder in VS Code. Open `.ipynb` file with seminar and select notebook
kernel (top right corner) as project's `.venv`. Possibly VS Code will ask to
install extensions for Jupyter.
